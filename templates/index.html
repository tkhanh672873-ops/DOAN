Mô hình AI

3 mô hình Machine Learning

Sử dụng và so sánh nhiều thuật toán phân loại để đạt kết quả tối ưu.

📈

Logistic Regression

Mô hình tuyến tính kinh điển, huấn luyện nhanh và cho kết quả ổn định, dễ diễn giải xác suất phân loại.

📊

Naive Bayes

Thuật toán xác suất Bayesian tối ưu cho phân loại văn bản, đặc biệt hiệu quả với đặc trưng TF-IDF.

⚡

Linear SVM

Support Vector Machine tuyến tính — mạnh mẽ với không gian đặc trưng thưa thớt, thường đạt độ chính xác cao nhất.

🔤

TF-IDF Bigrams

Vector hóa văn bản bằng TF-IDF kết hợp bigrams tối đa 5000 đặc trưng, phù hợp với ngôn ngữ tiếng Việt.
